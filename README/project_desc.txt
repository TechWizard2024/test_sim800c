J'ai 3 modules SIM800C USB (avec une carte SIM insérée dans chaque module SIM800C USB) connecté à mon pc windows (sur les ports COM) via un USB Hub 3.0 alimenté.
Je veux le code pour une application (Frontend web + Backend en GoLang + Base de données MyQSL) qui communique en temps reel avec les modules et permet de superviser et d'executer les fonctions des modules SIM800C USB

Le code généré sera stocké dans C:\xampp\htdocs\aa_Toolbox\test_sim800c\blackbox\v1

Le fichier Codes_USSD_CI.xlsx (voir pièce jointe ou dans C:\xampp\htdocs\aa_Toolbox\test_sim800c\storage\excel\) contient la liste des codes ussd des operateurs de telephonie mobile en Cote d'Ivoire (CI).
Le fichier contient les colonnes ci-dessous : 
	- ID : identifiant du code ussd
	- Carrier : Operateur executant le code ussd
	- Action : Categorie ou Registre dand lequel l'on peut classer l'action du code ussd 
	- Target : Interne si le beneficiare de l'action du code ussd est l'utilisateur de la SIM ou le numero de la SIM; 
				Externe si le beneficiare de l'action du code ussd est un utilisateur autre que celui de la SIM ou le beneficiaire est un numero autre que le numero de la SIM
	- Operation : Nom de l'Operation du code ussd
	- USSD_Code : Structure du code ussd
	- Information_INPUT : Information attendu en entrée avant execution du code ussd
	- Information_OUTPUT : Information obtenu en sortie après execution du code ussd
	- Scope : Defini si le code ussd en consideré comme utile (IN) ou inutile (OUT) dans le contexte du projet
	- Comment : Commentaire
	- Parent_USSD_ID : ID du code ussd parent du code ussd actuel


L'indicatif telephonique de la Cote d'Ivoire (CI) est +225 ou 00225 ou 225
Le plan de numérotation en Cote d'Ivoire (CI) est à 10 chiffires et est reparti comme suit par operateur de telephonie mobile : 
	- Orange CI : Préfixe 07 (ex: 07 XX XX XX XX)
	- MTN CI : Préfixe 05 (ex: 05 XX XX XX XX)
	- Moov Africa CI : Préfixe 01 (ex: 01 XX XX XX XX)
